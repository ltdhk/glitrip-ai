# Glitrip Planner - 更新日志

## [版本 1.0.0] - 2025-10-02

### ✨ 新功能

#### 1. 完整的旅行物品模板系统
- 新增 **70+ 种** 日常旅行常用物品模板
- 按类别精心分类：
  - 🧥 **服装类** (12项): T恤、长袖衬衫、长裤、短裤、内衣、袜子、运动鞋、拖鞋、外套、睡衣、泳衣、帽子
  - 📱 **电子产品类** (9项): 手机、充电器、充电宝、充电线、转换插头、耳机、相机、平板、笔记本电脑
  - 🧴 **洗护用品类** (12项): 牙刷、牙膏、洗面奶、洗发水、护发素、沐浴露、护肤品、防晒霜、化妆品、剃须刀、毛巾、梳子
  - 💊 **医疗健康类** (9项): 常用药品、感冒药、止痛药、创可贴、消毒液、肠胃药、晕车药、口罩、体温计
  - 👓 **配饰类** (6项): 太阳镜、手表、围巾、手套、腰包、雨伞
  - 📚 **书籍类** (3项): 旅行指南、笔记本、笔
  - 🎮 **娱乐类** (2项): 扑克牌、便携游戏机
  - 🎒 **其他类** (17项): 护照、身份证、钱包、现金、银行卡、行李箱、背包、保温杯、塑料袋、纸巾、湿巾、指甲刀、缝纫包、洗衣袋等

- 每个物品都标注了：
  - ✅ 是否为必需品
  - 🔢 建议携带数量
  - 📦 默认打包状态

  - 🇨🇳 中文：轻松旅行

### 📝 技术实现

#### 数据库结构
```
packing_items 表
├── destination_id = NULL  → 模板物品（可复用）
└── destination_id = 具体ID → 特定行程物品
```

#### 使用方式
1. 用户创建新的旅行目的地
2. 从模板中选择需要的物品
3. 一键添加到当前行程的打包清单
4. 可自定义修改数量和标记打包状态

### 🚀 部署说明

#### 对于新用户
- 首次启动 APP 会自动创建包含 70+ 物品的模板库
- 可以立即使用模板创建打包清单

#### 对于现有用户
- 升级后会自动触发数据迁移（数据库 v6 → v7）
- **⚠️ 注意**：所有示例数据将被清空，包括：
  - 目的地（北京、上海、成都等）
  - 文档数据
  - 物品数据（包括关联到目的地的物品）
  - 旅伴数据
- 用户配置（用户名、签名等）会保留
- 自动初始化 70+ 个模板物品
